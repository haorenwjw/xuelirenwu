#### 表格及列表  
|序号|技巧|功能|
|:-|:-:|:-:|
|1|AND|默认规则检索词之间用AND连接|
|2|+、-规则|包含或者不包含某些内容|
|3|不区分大小写|无|
|4|排除标点符号|无|
|5|词序和邻近|按照关键词的词序搜索|
|6|双引号|比较精确|
|7|书名号|书名号会出现在搜索内容中或者是被书名号括起来的内容不会被拆分|
|8|通配符*|表示匹配用它代表的任何词|
